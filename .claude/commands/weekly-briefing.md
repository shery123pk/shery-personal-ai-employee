Generate a CEO weekly strategic briefing.

## Steps

1. Run `scripts/briefing_generator.py weekly` to generate the weekly briefing
2. The briefing includes: Week Summary, Key Achievements, Pending Items, Strategic Recommendations
3. Output is saved to `AI_Employee_Vault/Briefings/weekly_briefing_YYYY-MM-DD.md`
4. Display the briefing contents to the user

Uses the OrchestratorAgent for LLM-powered strategic analysis.
